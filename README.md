# projeto_edl
## Projeto da disciplina de Estrutura de Dados Lineares

### Para compilar (comandos para linux/unix, com gcc instalado):

Abrir um terminal e executar:

```bash
git clone https://github.com/guilarre/projeto_edl.git
cd projeto_edl
gcc funcoes_hash.c -o funcoes_hash -lm
./funcoes_hash
```
